## 🚀 How to Use

### Text Formatting
```
1. Select any text
2. Bubble menu appears automatically
3. Use any control:
   - Font dropdown → Change font
   - Size dropdown → Change size
   - Color icon → Pick color
4. Changes apply instantly
5. Deselect → menu disappears
```

5. Action executes or modal opens
